To actually use it, the typepath of the object has to be contained within the sanity datum valid_inspiration list. Assign this component to an item specifying which statistics should be levelled up, and the item will be able to be used as an inspiration. The format of statistics is list(STAT_DEFINE = number) or a proc that returns such a list.
